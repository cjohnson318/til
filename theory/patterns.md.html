# Patterns Collect notes about patterns from the Gang of Four. ## Creational - Abstract Factory - Prototype ### Builder Create complex object in multiple steps. ### Factory Method Configure new object at runtime. ### Singleton Limit creation of class to one object. ## Structural - Bridge - Composite - Decorator - Facade - Flyweight - Proxy ### Adapter Adapt interface of one object to another. ## Behavioral - Chain of Responsibility - Command - Interpreter - Mediator - Memento - Template Method - Visitor ### Observer When one item changes, then dependencies are updated. ### State Hold all states. Control transition from one state to another. ### Strategy Select algorithm at runtime.
